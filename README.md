# Sistema de Cadastramento de Funcionários

Este projeto é um **programa de cadastramento de funcionários** desenvolvido em **Java** utilizando **interface gráfica com Swing**.  
O sistema tem como objetivo permitir o registro e a visualização de informações de funcionários de forma simples, organizada e prática.

## 📌 Objetivo

O projeto foi desenvolvido com a finalidade de aplicar conceitos de:

- Programação Orientada a Objetos em Java
- Desenvolvimento de interfaces gráficas com Swing
- Manipulação de dados em formulários
- Organização de um sistema simples de cadastro

## ✨ Funcionalidades

- Cadastro de funcionários
- Inserção de dados por meio de interface gráfica
- Exibição das informações cadastradas
- Interface simples e intuitiva
- Estrutura organizada para fins de estudo e aprendizado

## 🛠️ Tecnologias utilizadas

- **Java**
- **Swing**
- **NetBeans**
